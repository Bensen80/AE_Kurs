### Übungen zu Wrapper Klassen
1. Erstellen Sie ein prozedurales Programm, welches vom Benutzer eingegebene Daten als String einliest
  * Methode: einlesen()

2. Es soll überprüft werden, ob der eingegebene Wert eine Ganzzahl ist
  * Methode: isNumeric()

3. Ist der Wert eine Ganzzahl soll er in den gewünschten Datentyp __int__ konvertiert werden
  * Methode: convert()
  
4. Ist eine Konvertierung nicht möglich, soll der Benutzer darüber informiert werden und erneut die Möglichkeit zur EIngabe haben.

5. Über eine Ausgabe Methode soll der Wert ausgegeben werden
  * Methode: ausgabe()
  
6. Testen Sie ihr Programm in der main() Methode
